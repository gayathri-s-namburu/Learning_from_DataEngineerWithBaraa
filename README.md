# Phase 1 - The Foundations 🌱

# SQL

- SQL is the language used to work with structured data stored in databases.
- In data engineering, SQL is your **main tool for transforming data** inside warehouses and lakehouses.

---

## 🔥 Topics to Learn:

- [ ]  Basic Queries (`SELECT`, `WHERE`, `ORDER BY`, `LIMIT`, `GROUP BY`, `HAVING`)
- [ ]  Filtering & Conditions (`AND`, `OR`, `IN`, `BETWEEN`, `LIKE`)
- [ ]  Data Transformations and Cleaning: String, Number, Date, Null Functions
- [ ]  Case Statements `CASE WHEN`
- [ ]  Aggregations (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
- [ ]  Joins (`INNER`, `LEFT`, `RIGHT`, `FULL`)
- [ ]  Window functions (basic usage like `ROW_NUMBER`, `RANK`)
- [ ]  Writing clean, readable, and maintainable SQL (CTE, Subqueries, Views, CTAS, Temp Tables )

# Python

- Python is a **general-purpose programming language** known for being simple, readable, and powerful.
- In data engineering, **Python is your engine** - it automates tasks, transforms data, builds workflows, and connects tools.

  ## 🔥 Topics to Learn:

- [ ]  Variables & Data Types
- [ ]  Data Structures (lists, tuples, dictionaries, sets)
- [ ]  Control Flow (if/else, loops)
- [ ]  Functions (reusable logic)
- [ ]  Working with Files (CSV, JSON, text)
- [ ]  Error Handling (try/except)
- [ ]  Logging (production-ready code)
- [ ]  Connecting to Systems (databases, APIs, Kafka basics)
